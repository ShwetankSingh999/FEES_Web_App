# FEES Evaluation Web App

A professional, client-side web application for **Fiberoptic Endoscopic Evaluation of Swallowing (FEES)**. This tool allows clinicians to record patient information, document pre-swallow assessments, and perform swallow trials with automated therapy suggestions and PDF report generation.

## 🚀 Key Features

- **Step-by-Step Clinical Workflow**: Guided assessment from patient info to final clinical impressions.
- **Dynamic Swallow Trials**: Add and manage multiple trials with PAS (Penetration-Aspiration Scale) scoring.
- **Automated Therapy Suggestions**: Get suggested exercises and compensatory strategies based on evaluation findings.
- **Professional PDF Reports**: Generate a clean, branded clinical report for medical documentation (now fixed for special character compatibility).
- **Privacy-First**: Entirely client-side application. No data is sent to a server; all information stays in your browser's local storage or in the generated PDF.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3 (Glassmorphism & Modern UI)
- **Logic**: Vanilla JavaScript (ES6+)
- **PDF Generation**: [jsPDF](https://github.com/parallax/jsPDF)

## 📦 Local Installation

1. Clone the repository:
